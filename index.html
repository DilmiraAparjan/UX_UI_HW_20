Problem Statement

Friends of Animals was designed as a non-profit to help animals in need and to spread awareness. We have observed that the Friends of Animal website's navigation and design is making it hard for users to find important information which is causing users to give up on learning more about the site. How might we help Brenda, an avid animal advocate, improve her user experience when navigating the Friends of Animals website?

Iterations

Our early designs that we created focused on the navigation more than the content. We added some artistic elements like a squiggly line throughout our design.
However, through user testing, we realized that a lot of our users were not fond with the squiggly line because of how distracting it was. Some users also commented on how it was hard to differentiate the different navigation tabs because it was in a gray scale
